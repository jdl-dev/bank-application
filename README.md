Backend application of a banking system created as a portfolio project.

## Project goal
The goal of this project is to build a realistic banking backend that demonstrates:
- clean architecture
- authentication and authorization
- database design
- transactional business logic
- security
- testing
- Docker-based local environment

## Planned modules
- authentication
- users
- customer profiles
- bank accounts
- transfers
- beneficiaries
- admin panel
- audit logs

## Tech stack
- Java 21
- Spring Boot
- Spring Security
- Spring Data JPA
- PostgreSQL
- Flyway
- Validation
- Lombok
- Testcontainers
- Docker Compose

## Project status
Project initialization and architecture planning in progress.
